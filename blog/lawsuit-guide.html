第一步：起诉准备

1. 能不能起诉？

先判断你的情况属于法院管辖范围。法院受理的是民事纠纷——比如欠钱不还、合同违约、离婚、房产纠纷、侵权赔偿等。刑事案件由检察院公诉，行政纠纷走行政诉讼，不归民事法庭管。

2. 去哪里起诉？

民事诉讼法最基本的原则是"原告就被告"——到被告住所地的人民法院起诉。比如对方是广州人，就去广州的区法院。合同纠纷还可以选择合同履行地法院。借贷纠纷，出借人所在地法院也有管辖权。

简单说：被告在哪，就去哪的法院告。

3. 起诉需要什么材料？

这是最关键的。去法院立案，必须带齐以下三样：

材料名称说明
起诉状写明原告、被告基本信息、诉讼请求（你想要什么结果）、事实与理由（为什么告他）。一式三份（法院一份、对方一份、自留一份）。
身份证明原告身份证复印件。如果是个体户或公司，还要营业执照复印件和法定代表人身份证明。
证据材料能证明你主张事实的材料：合同、借条、转账记录、微信聊天记录、录音、照片、证人联系方式等。证据要整理好，编上序号。
⚠ 经验提醒：很多人起诉失败是因为起诉状写得不规范——诉讼请求不明确、事实陈述不清晰。建议写好后找律师看一眼。我经常帮当事人修改起诉状，省去来回跑法院补正的麻烦。

4. 怎么算诉讼费？

诉讼费按争议金额计算。比如1万元以下诉讼费50元；10万元的案件诉讼费约2300元。一般由原告先预交，败诉方承担。金额不大可以自己算，登录"广东法院诉讼服务网"有诉讼费计算器。

第二步：去法院立案

1. 立案方式

现在立案有两种方式：

  • 现场立案：带齐材料到法院诉讼服务中心（立案大厅），窗口工作人员审查后符合条件的当场立案。
  • 网上立案：通过"广东法院诉讼服务网"或微信小程序"粤公正"提交电子材料，审查通过后再邮寄纸质材料。

2. 立案审查多久？

法律规定7日内决定是否立案。实践中：

  • 材料齐全、事实清楚 → 当天或3天内出立案通知书
  • 材料不齐 → 法院出具《补正通知书》，按要求补正后重新计算审查期
  • 不符合起诉条件 → 法院出具《不予受理裁定书》，你不服可以上诉

3. 立案后会发生什么？

立案后，法院会发给你《案件受理通知书》和《举证通知书》。法院会把起诉状副本、证据副本和开庭传票寄送给被告。被告收到后有15天答辩期（民事案件）。

💡 小技巧：立案时如果知道对方可能转移财产，可以同时申请财产保全——查封、冻结对方银行账户或房产。财产保全是打官司中非常有效的策略，能让你在判决后顺利执行到钱。

第三步：等待开庭

立案后到开庭前这段时间，不是干等着。有几件事很重要：

1. 举证期限

法院会指定举证期限，一般15-30天。所有证据必须在这个期限内提交，超期提交法院可以不采纳。所以证据要早准备、早提交。

2. 对方答辩

收到对方寄来的答辩状后，要仔细看对方的答辩理由，预判对方会怎么反驳，提前准备应对思路。

3. 是否申请调解？

现在法院普遍推行诉前调解——立案前先由调解员组织双方调解。作为有6年法院调解员经验的人，我的观点是：如果双方差距不大，调解是最快、最省钱的方案。但如果对方根本没有诚意，就没必要浪费时间，直接进入诉讼程序。

4. 开庭时间

简易程序案件一般在立案后1-2个月内开庭。普通程序案件会慢一点，2-3个月左右。具体看法院排期，广州的区法院案件量大，排期相对较长。

第四步：开庭审理

很多当事人第一次开庭非常紧张，其实开庭有固定的流程，提前了解就不慌了。

开庭流程

环节内容
① 庭前准备书记员查明当事人是否到庭，宣读法庭纪律。法官入庭，核对当事人身份。
② 法庭调查原告宣读起诉状 → 被告答辩 → 原告举证 → 被告质证 → 被告举证 → 原告质证 → 法官询问。
③ 法庭辩论围绕争议焦点双方轮流发表意见。注意：不要说与案件无关的话，不要人身攻击。
④ 最后陈述简单概括你的诉求。比如："请求法院支持我全部诉讼请求。"一句话就行。
⑤ 调解法官会询问双方是否愿意调解。如果能当庭达成调解协议，当场出调解书，具有法律效力。
⑥ 宣判可以当庭宣判，也可以择日宣判。择日宣判的话，法院会通知你领取判决书。
⚠ 开庭注意事项：
• 按时到庭！迟到超过30分钟可能按撤诉处理（原告）或缺席判决（被告）。
• 带齐所有证据原件（开庭时要核对）。
• 说话简单直接，回答法官问题，不要讲与本案无关的事。
• 不要激动，不要当庭吵架。法官不喜欢情绪化的当事人。
• 不懂的可以问：法官或律师可以提醒你。

第五步：判决与执行

1. 拿到判决书后

判决书会写明：判决结果、事实认定、法律依据、诉讼费承担。注意看判决书最后一页——上诉期限。不服一审判决，可以在收到判决书之日起15日内向上一级法院上诉。不上诉，判决在15天后生效。

2. 判决生效后对方不履行怎么办？

判决生效后，对方如果拒不履行（比如不还钱、不过户、不交房），你需要向法院申请强制执行。申请执行的期限是2年（从判决生效之日起算）。

法院执行措施包括：

  • 查询、冻结、划扣被执行人银行存款
  • 查封、拍卖被执行人房产、车辆
  • 限制高消费、列入失信被执行人名单（"老赖"名单）
  • 司法拘留（情节严重）

3. 打官司从开始到结束要多久？

程序类型审限实际周期（含立案、送达等）
简易程序3个月一般1-4个月
普通程序6个月一般3-9个月
二审（上诉）3个月一般1-3个月

一个案子如果一审+二审全部走完，长的可能要一年左右。

借条到期了但对方不还钱，起诉来得及吗？

要看诉讼时效。民间借贷的诉讼时效是3年，从知道权利受侵害之日起算。如果借条写了还款日期，从还款日到期那天开始算3年。建议尽快行动，一旦超过诉讼时效，法院可能不支持你的诉讼请求。如果快过期了，可以先让律师帮忙发律师函中断时效。电话：13129376586。

我想离婚但对方不同意，起诉离婚怎么操作？

一方不同意离婚，只能走诉讼离婚。流程：写起诉状（写明结婚时间、离婚理由、有无子女、财产情况、你的诉求）→ 到被告住所地法院立案 → 法院安排调解（我是越秀区法院调解员，这类案件我调解过几百件）→ 调解不成转开庭 → 判决。法院第一次通常判不离，但判决生效6个月后再次起诉，第二次判离的可能性很大。离婚案件材料比较复杂，建议咨询律师，我的电话：13129376586。

我的几点忠告

做了13年律师，和几千个当事人打过交道，有几点我想特别说：

  1. 打官司不是最好的解决方式。能协商解决尽量协商。诉讼费钱费时还伤和气。我在法院做了6年调解员，见过太多打了两年官司、花了十几万律师费，结果跟最初调解方案差不多的案子。
  2. 证据是关键。打官司就是打证据。再有理没有证据，法院也不会支持你。日常注意保留合同、转账记录、聊天记录等证据。
  3. 不要错过期限。诉讼时效、举证期限、上诉期限——这些时间点一旦错过，神仙也救不了。如果你记不住这些期限，最好找个律师。
  4. 不要在网上乱搜着学。网上很多法律知识是过时的或片面的。民法典2021年实施后很多规定改了，网上有些旧文章引用的是旧法，照着做可能要吃亏。
